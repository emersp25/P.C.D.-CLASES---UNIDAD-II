01
El problema de la memoria

¿Qué es un Dominio?

Las computadoras en internet se identifican entre sí usando números llamados Direcciones IP. Cada servidor, cada página web, tiene una dirección numérica única asignada.

🖥️
Lo que entiende la máquina
142.250.190.46
Dirección IP de Google
↔️
DNS traduce
👤
Lo que recuerda el humano
google.com
Dominio registrado
🗺️
La metáfora de las coordenadas

Imaginar tener que memorizar las coordenadas GPS exactas de la casa de cada amigo para poder visitarlo. Sería imposible. Por eso usamos direcciones como "Avenida 1, Calle 2, San Lucas".

Un Dominio de Internet cumple la misma función: es una traducción legible por humanos de una dirección IP numérica. El sistema que realiza esa traducción se llama DNS (Domain Name System).

⚡
Cada vez que se escribe una dirección web en el navegador, el DNS consulta su base de datos en milisegundos y traduce el nombre al número IP correspondiente — todo antes de que la página empiece a cargar.

Estructura técnica

Anatomía de un Dominio Web

Una dirección web no es un bloque de texto — tiene partes bien definidas, cada una con una función específica. Se analiza el siguiente ejemplo real:

www.mineduc.gob.gt
Subdominio División dentro del dominio. Hoy casi no es obligatorio escribirlo.
Dominio de 2º nivel El nombre elegido por la organización. Es lo que se registra.
Extensiones (.gob.gt) Indican la naturaleza del sitio y su ubicación geográfica.
🏷️

Subdominio

Permite crear secciones dentro del mismo sitio. Ejemplo: blog.empresa.com o tienda.empresa.com.

✏️

Segundo Nivel

El nombre de la marca. Es único en el mundo para su extensión — si alguien ya lo registró, no se puede usar.

🔖

Extensión (TLD)

Top-Level Domain. Clasifica el sitio por naturaleza (.com, .org) o por país (.gt, .mx).

03
gTLD — Genéricos

Extensiones por Naturaleza

Las extensiones genéricas indican a qué se dedica o a quién pertenece un sitio. Algunas son libres; otras están estrictamente restringidas.

.com

Comercial

El rey indiscutible de internet. Cualquier empresa o persona puede registrarlo. Es el primero que el usuario busca.

🟢 Libre — cualquiera puede registrarlo
.org

Organización

Creado para entidades sin fines de lucro: ONGs, iglesias, fundaciones y asociaciones comunitarias.

🟢 Libre — pero asociado a causas sociales
.edu

Educación

Reservado estrictamente para escuelas y universidades avaladas por las autoridades de educación.

🔴 Restringido — no se puede comprar libremente
.gob / .gov

Gobierno

Uso exclusivo para instituciones del Estado. Solo una entidad gubernamental puede registrarlo.

🔴 Restringido — solo para el Estado
.net

Network

Creado para empresas de telecomunicaciones. Hoy se usa como alternativa cuando el .com ya está ocupado.

🟢 Libre — alternativa al .com
💡 Regla de seguridad: Una página de gobierno que termine en .com o .org es una señal de alerta inmediata. Los trámites oficiales siempre usan extensiones restringidas.

El impacto local en el SEO

Al usar un dominio .gt o .com.gt, se le indica a Google: "Este negocio opera en Guatemala". Esto hace que el sitio aparezca primero cuando alguien en el país busca el servicio, en lugar de competir con negocios de España o México que usan el mismo idioma.

El Negocio de los Registradores

⚠️ El mito: "Voy a comprar un dominio para siempre"

La realidad es que los dominios se alquilan, normalmente por año. Empresas como GoDaddy, Namecheap y Hostinger funcionan como bienes raíces digitales: cobran una tarifa anual para mantener el nombre reservado a nivel mundial.

GoDaddy Namecheap Hostinger Google Domains
⚠️ Si no se renueva el dominio antes de que venza, cualquier otra persona puede comprarlo y el sitio queda sin dirección.

Actividad en clase

El Detective de Fraudes

El phishing es una técnica de estafa que usa dominios falsos diseñados para parecerse a sitios reales. Conocer la anatomía de un dominio es la primera línea de defensa.

Para cada par de enlaces, el estudiante identifica cuál es real y cuál es una estafa, basándose únicamente en la lógica del dominio y su extensión.

Caso 1 🏦 El Banco
A www.banco-industrial-seguro.com
B www.bi.com.gt
📋 Veredicto — B es el sitio real

Las instituciones financieras locales serias usan la extensión del país (.com.gt) para validar su operación en Guatemala. El dominio A (.com genérico) puede ser registrado por cualquier persona en el mundo en minutos por pocos dólares — es la herramienta favorita de los estafadores.

Caso 2 🏛️ El Gobierno
A www.sat.gob.gt
B www.tramites-sat-guatemala.org
📋 Veredicto — A es el sitio real

Solo el Estado puede usar la extensión .gob. Ningún trámite oficial del gobierno guatemalteco estará alojado en un .org. El dominio B combina palabras que suenan oficiales ("trámites", "SAT", "Guatemala") para engañar visualmente, pero la extensión delata que no es gubernamental.

Caso 3 🎓 La Universidad
A www.usac.edu.gt
B www.inscripciones-usac.com
📋 Veredicto — A es el sitio real

La extensión .edu está restringida: solo las instituciones académicas oficiales pueden usarla. Si las inscripciones de la USAC fueran en un .com, cualquier particular podría haber comprado ese dominio. Ante cualquier proceso de inscripción universitaria, siempre se debe verificar que el dominio use .edu.

🛡️ Reglas para detectar phishing por el dominio
Verificar que la extensión corresponda a la naturaleza del sitio (.gob para gobierno, .edu para educación).
Desconfiar de dominios largos con palabras como "seguro", "oficial", "bono" o el nombre de la institución dentro del dominio de segundo nivel.
Una institución seria nunca pedirá datos personales en un dominio .com genérico.
